# Caderno Temático com NotebookLM
## Educação Financeira para Iniciantes

## 📌 Objetivo

Este projeto demonstra o uso do NotebookLM como ferramenta de aprendizagem ativa.

O objetivo foi reunir diferentes fontes confiáveis sobre educação financeira e utilizar IA para:

- resumir conteúdos;
- identificar conceitos importantes;
- responder perguntas estratégicas;
- criar um guia de revisão;
- produzir prompts reutilizáveis.

---

# Objetivos de Estudo

Ao finalizar este estudo pretendo compreender:

- O que é educação financeira;
- Diferença entre ativos e passivos;
- Conceitos de orçamento pessoal;
- Reserva de emergência;
- Juros simples e compostos;
- Investimentos básicos.

---

## 📚 Fontes Utilizadas

Os seguintes materiais oficiais do Banco Central do Brasil foram utilizados como base para o NotebookLM e estão disponíveis na pasta `/fontes` deste repositório:

1. `caderno_cidadania_financeira.pdf`
   - Introdução aos conceitos de cidadania e educação financeira.

2. `conceito_cidadania_financeira.pdf`
   - Definição e importância da cidadania financeira.

3. `glossario_cidadania_financeira.pdf`
   - Glossário oficial com os principais termos financeiros.

4. `SerieCidadania_2educ_fin_funciona.pdf`
   - Estudo sobre a efetividade da educação financeira.

Todos os documentos são de domínio público e foram obtidos no portal oficial do Banco Central do Brasil.

## 🧠 Engenharia de Prompts

Durante o estudo foram realizados diferentes testes de prompts para explorar as capacidades do NotebookLM.

### Prompt 1
**Objetivo:** obter uma visão geral do conteúdo.

```
Faça um resumo geral do conteúdo presente nas quatro fontes em até 15 tópicos.
```

**Resultado**

Foi produzido um resumo estruturado destacando educação financeira, planejamento financeiro, orçamento, consumo consciente, investimentos e cidadania financeira.

---

### Prompt 2
**Objetivo:** identificar conceitos comuns entre as fontes.

```
Quais são os principais conceitos presentes em todas as fontes?
```

**Resultado**

A IA organizou os conceitos em formato de tabela, indicando em quais documentos cada conceito estava presente.

---

### Prompt 3
**Objetivo:** criar um glossário.

```
Crie um glossário contendo os principais termos financeiros.
```

**Resultado**

Foi gerado um glossário com definições simples e objetivas para apoiar futuras revisões.

---

### Prompt 4
**Objetivo:** simplificar a linguagem.

```
Explique os conceitos utilizando linguagem simples.
```

**Resultado**

As respostas ficaram mais acessíveis para iniciantes, facilitando o entendimento dos conteúdos.

---

### Dificuldades Encontradas

Os primeiros testes utilizaram prompts muito genéricos, resultando em respostas amplas. Após tornar os comandos mais específicos e solicitar o uso exclusivo das fontes carregadas, as respostas ficaram mais organizadas, objetivas e alinhadas ao objetivo do estudo.

# Miniguia de Estudo

## Resumo

A educação financeira consiste na capacidade de administrar recursos financeiros de forma consciente, planejando gastos, economizando, formando reservas e investindo para alcançar objetivos futuros.

Os principais pilares são:

- orçamento;
- controle financeiro;
- consumo consciente;
- reserva de emergência;
- investimentos;
- planejamento de longo prazo.

---

## Glossário

| Conceito | Definição |
|----------|-----------|
| Orçamento | Planejamento das receitas e despesas |
| Ativo | Bem ou investimento que pode gerar valor |
| Passivo | Dívida ou obrigação financeira |
| Reserva de Emergência | Dinheiro destinado a imprevistos |
| Juros Compostos | Juros calculados sobre o valor acumulado |
| Investimento | Aplicação de recursos visando retorno futuro |

# Prompts Reutilizáveis

- Faça um resumo do conteúdo utilizando apenas as fontes carregadas.
- Explique o assunto em linguagem simples.
- Crie um mapa mental do tema.
- Gere um glossário com os principais conceitos.
- Faça cinco perguntas para revisão.
- Monte um plano de estudos de uma semana.
- Compare os conceitos apresentados pelas diferentes fontes.
- Cite em qual fonte cada informação foi encontrada.
